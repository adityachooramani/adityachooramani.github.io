HoneyNet-IoT is an embedded research system designed to trap and monitor malicious probes targeting Internet-of-Things (IoT) devices. By deploying a decoy network that mimics legitimate IoT services, this project allows for the detailed observation of attacker behavior in a controlled environment.

üèóÔ∏è System Architecture#

The project utilizes a low-cost, hardware-centric approach to cybersecurity research.

Core Components#

  • Decoy Node: An Arduino Uno R3 acting as a low-interaction honeypot. It emulates common IoT protocols (e.g., HTTP, Telnet) to lure attackers.
  • Network Interface: Integrates with network hardware to expose emulated services to the local network or internet.
  • Logging Backend: A centralized server that intercepts logs sent by the Arduino, persisting data for later analysis.

üõ†Ô∏è Key Engineering Features#

  • Service Emulation: Implemented lightweight service responders that mimic real IoT device replies, increasing attacker engagement.
  • Telemetry Capture: Built a pipeline to record critical attacker metadata, including Source IP, Port, Payloads, and Timestamps.
  • Resource Optimization: Optimized the Arduino firmware to handle basic network interactions while remaining within the strict memory limits of the ATmega328P microcontroller.
  • Attack Diversion: Designed to reduce risk to production systems by providing a high-visibility target for reconnaissance and exploitation attempts.

üî¨ Research Value & Impact#

  • Threat Intelligence: Gathers real-world data on common IoT exploit vectors and reconnaissance patterns.
  • Vulnerability Study: Identifies evolving tactics used by adversaries to compromise smart devices.
  • Cost-Effective Defense: Demonstrates how $20 hardware can be used to build effective security research tools.
